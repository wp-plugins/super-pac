Help Colbert Super PAC rank No.1 in Google for 'Super PAC'

== Description ==

Shortcode to add a link to http://www.colbertsuperpac.com/ along with Stephen's description of Super PAC. More links back to the Super PAC website should push it into the number one position for the search 'Super PAC'.

Use the shortcode [superpac] one or more times in pages and posts on your WordPress site and display the following:

[Super PAC](http://www.colbertsuperpac.com/ "Super PAC")

A frothy mix of lube and campaign funding that is sometimes the byproduct of politics

== Installation ==

The automatic plugin installer should work for most people.

To manually install the plugin:

1. Upload the plugin folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
